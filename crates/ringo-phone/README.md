# ringo

A terminal SIP softphone built on [baresip](https://github.com/baresip/baresip):
manage multiple SIP profiles, place and receive calls from a ratatui TUI, and
drive a running session remotely (`ringo control`) or headless for automation.

```sh
cargo install ringo-phone   # installs the `ringo` binary
```

Full documentation — features, installation, keybindings, remote control and
configuration — lives in the project README:
**<https://github.com/davidborzek/ringo#readme>**
