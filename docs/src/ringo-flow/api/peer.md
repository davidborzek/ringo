## peer.name

**Receiver** [`Peer`](peer.md) · **Returns** `string?`

<a id="number"></a>

## peer.number

**Receiver** [`Peer`](peer.md) · **Returns** `string?`

## peer.uri

**Receiver** [`Peer`](peer.md) · **Returns** `string?`

The remote party's full URI (e.g. `sip:bob@example.com`), or `()`.
